Building Intelligent AI Agents with Microsoft Foundry
Overall Estimated Duration: 6 Hours
Welcome to the Intelligent AI Agents Hack in a Day! In this hands-on experience, you will learn how to design and build goal-driven AI systems using Microsoft Foundry, Azure AI Search, and modern AI agent frameworks. Throughout this, you will develop intelligent applications that combine generative AI, Retrieval-Augmented Generation (RAG), AI agents, observability, and multi-agent collaboration to solve real-world problems.

Scenario
Contoso Wellness, a digital health and fitness platform, wants to provide users with an intelligent AI assistant capable of answering health-related questions, analyzing nutrition data, retrieving wellness resources, and generating personalized fitness recommendations. Traditional chatbots are limited because they rely only on static responses or pre-defined workflows.

To improve the user experience, Contoso Wellness plans to build an AI-powered multi-agent system that can:

Understand user health and fitness questions
Retrieve relevant knowledge from trusted sources
Perform calculations such as BMI analysis
Access real-time web information
Collaborate across specialized AI agents to generate insightful responses
By integrating Microsoft Foundry, AI models, vector search, and agent frameworks, the organization aims to create a scalable AI platform capable of delivering personalized and reliable health guidance.

Introduction
Your mission is to build an AI-powered Health & Fitness Assistant using Microsoft Foundry and agent frameworks. During this Hack in a Day, you will progressively develop an AI system capable of reasoning, retrieving knowledge, and coordinating multiple specialized agents.

Using Microsoft Foundry SDK, Azure AI Search, and Visual Studio Code, you will design an end-to-end solution that can:

Initialize an AI development environment and deploy models
Generate chat completions and embeddings using AI models
Build Retrieval-Augmented Generation (RAG) pipelines for knowledge retrieval
Develop specialized AI agents for health and wellness assistance
Integrate search tools and real-time web grounding
Implement observability and tracing for monitoring AI systems
Build multi-agent workflows capable of collaborative reasoning
This solution demonstrates how modern AI systems combine language models, retrieval systems, and autonomous agents to build intelligent applications.

Learning Objectives
By participating in this Hack in a Day, you will learn how to:

Set up and configure a Microsoft Foundry AI development environment
Deploy and interact with AI models for chat completions and embeddings
Implement Retrieval-Augmented Generation (RAG) using Azure AI Search
Build and orchestrate AI agents using Microsoft Foundry SDK
Integrate external tools such as Bing Search and vector search
Implement observability and tracing for monitoring AI systems
Build multi-agent AI pipelines using Semantic Kernel and agent frameworks
Hack in a Day Format: Challenge-Based
This hack is structured into six progressive challenges that represent the lifecycle of building a real-world AI assistant:

Challenge 01: Create the Microsoft Foundry AI Environment
Provision core Azure resources, configure the AI hub and project, and deploy the required AI models.

Challenge 02: Initialize the AI Project and Run Your First Model
Configure the development environment, authenticate credentials, and test AI model interactions.

Challenge 03: Build a Retrieval-Augmented Generation (RAG) Assistant
Implement embeddings, create a search index, and generate knowledge-grounded responses using vector retrieval.

Challenge 04: Develop Specialized AI Agents
Build multiple AI agents capable of handling fitness guidance, BMI calculations, resource search, and web-grounded reasoning.

Challenge 05: Enable Observability and Monitoring
Implement tracing and observability to monitor AI agent performance using Azure Monitor and telemetry tools.

Challenge 06: Build a Multi-Agent AI System
Integrate Semantic Kernel and multi-agent frameworks to build collaborative AI systems capable of solving complex health queries.

Throughout each challenge, you will iteratively design, build, and test intelligent AI systems, evolving from a simple AI interaction to a fully functional multi-agent AI assistant.

Challenge Overview
You will begin by provisioning the Microsoft Foundry AI environment, including the necessary Azure resources and AI models. Next, you will configure the development environment and validate connectivity with the AI services.

After the environment is prepared, you will build a Retrieval-Augmented Generation pipeline, enabling the AI system to retrieve and reason over health-related knowledge.

You will then develop several specialized AI agents, each capable of handling different tasks such as answering wellness questions, calculating BMI, retrieving knowledge from documents, and accessing real-time web information.

To ensure reliability and transparency, you will implement observability and tracing, allowing you to monitor agent interactions and diagnose system behavior.

Finally, you will combine these capabilities into a multi-agent AI architecture, where multiple agents collaborate to provide intelligent responses and advanced reasoning.

By the end of this Hack in a Day, you will have a fully functional AI-driven Health & Fitness Assistant powered by Microsoft Foundry and modern AI agent frameworks.
